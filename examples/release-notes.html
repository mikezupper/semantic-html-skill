sx release notes

Newest first — the release list is an <ol reversed>, so the numbering counts down and machines still know the true order. Edits to published notes stay visible as <ins> and <del>, each stamped with a datetime.

  1. Version 2.4.0

    codename Hgroup Therapy

    Released July 20, 2026

    • New --strict flag: validator warnings become errors.
    • Ruby annotation output is enabled by default. Ruby annotation output is opt-in via --ruby — default output was breaking plain-text diffs.
    • Conversion is now twice three times as fast on large books — the earlier claim measured only the warm cache.

  2. Version 2.3.0

    codename Search Party

    Released May 4, 2026

    • Emit <search> for detected search forms.
    • Exclusive accordions: consecutive FAQs get <details name> groups.
    • Erratum: the minimum supported Java version for the bundled validator is 11, not 8.

  3. Version 2.2.0

    codename Time and Data

    Released February 11, 2026

    • Dates and durations in prose are wrapped in <time datetime> automatically.
    • Quantities with units become <data> elements.

Editing policy

Published notes are never silently rewritten. Corrections appear as <ins>/<del> pairs with a datetime and, where one exists, a cite URL pointing at the issue that prompted the change. A fact that merely became outdated — not wrong — is struck with <s> and corrected in place.
